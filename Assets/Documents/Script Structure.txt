Keep 'em Out Script Structure

Description and usage at the start of a script!

Namespace: Project.Managers (use Project.Managers.Properties if you need a class with stuff for Lists)
	GameManager
	KeyManager
	ResourceManager
	EventManager
	UIManager
	AudioManager	
	SaveManager
	ConfigurationManager
	TranslationManager

Namespace: Project.Player
	PlayerStats
	PlayerMovement
	PlayerLook
	PlayerAttack
	PlayerAnim
	PlayerUI

	PlayerStats -> PlayerMovement, PlayerLook, PlayerAttack, PlayerUI
	PlayerMovement & PlayerAttack -> PlayerAnim

Namespace: Project.Enemy
	EnemyStats
	EnemyMovement
	EnemyAnim
	EnemyAttack
	EnemyUI

	EnemyStats -> EnemyMovement, EnemyAttack, EnemyUI
	EnemyMovement & EnemyAttack -> EnemyAnim

Namespace: Project.Environment
	EnvStats (When needed like Breakable Wall HP)
	EnvUI (When needed like Breakable Wall HP)

Namespace Project.UI
	UIScripts (all individual)

	UIScripts -> EventManager, UIManager
	UIManager -> UIScripts
